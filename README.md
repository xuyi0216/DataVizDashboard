# 大屏项目 / DataVizDashboard

一个面向初学者的数据可视化大屏开源项目，带大家**从 0 到 1** 动手实现一个属于自己的数据可视化大屏。

## 项目简介

本项目旨在帮助零基础的同学，通过循序渐进的实战，掌握数据可视化大屏的核心技能，包括：

- 大屏布局与自适应（缩放、流式布局）
- 常用图表（折线、柱状、饼图、地图等）的实现
- 数据接入与实时刷新
- 主题样式与视觉效果设计

本项目会持续维护和更新，欢迎一起学习、共建。
